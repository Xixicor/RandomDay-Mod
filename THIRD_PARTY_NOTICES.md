# Third-Party Notices

RandomDay is designed to work with separately installed server-side mods through optional adapters.

Supported mods are not bundled in this repository or release package.

Optional adapter targets tested for this release:

- Chaotic Spawn Engine / DynamicSpawns by FalseLight / FalseLight27, tested with 0.8.5 Hotfix 1.
- Lethal Lasers by FalseLight / FalseLight27, tested with 1.0 UE4SS.
- ANTE Up by combatken, tested with 0.26.
