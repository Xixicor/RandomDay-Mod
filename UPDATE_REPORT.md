# RandomDay v0.3.15-alpha Repository Update Report

This package updates the uploaded GitHub repository to the current RandomDay public release structure.

## Main decisions

- Repository title remains `RandomDay-Mod`.
- GitHub URL remains `https://github.com/Xixicor/RandomDay-Mod`.
- Public mod name is `RandomDay`.
- Runtime install folder is `RandomDay/`.
- Release asset is `RandomDay_v0.3.15-alpha.zip`.

## Supported adapters

- DynamicSpawns / Chaotic Spawn Engine 0.8.5 Hotfix 1
- Lethal Lasers 1.0 UE4SS
- ANTE Up 0.26

Supported mods are not bundled. RandomDay only includes adapter scripts.

This update includes current pages and descriptions for the public release. Full per-family, per-day profile tables and complete SandboxSettings.ini references are intentionally reserved for the next documentation pass.
