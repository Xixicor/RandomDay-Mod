# Contributing

RandomDay is maintained as a controlled project.

Public pull requests, unsolicited code submissions, automated rewrites, generated-code cleanup submissions, and speculative feature patches are not accepted unless explicitly requested by the repository owner.

## Accepted contact

For bug reports, compatibility notes, or server-administration observations, use the contact channels listed in the project documentation.

## Not accepted

The following are not accepted by default:

- unsolicited pull requests;
- generated-code rewrites;
- broad refactors;
- style-only formatting changes;
- untested gameplay logic changes;
- bundled third-party mod files;
- redistributed third-party source code;
- changes that imply official affiliation with Abiotic Factor, UE4SS, Nexus Mods, or third-party mod authors.

## Controlled development

The repository owner may request specific review, testing, or documentation help when needed. Unrequested submissions may be closed without review.
