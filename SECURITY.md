# Security policy

RandomDay writes server configuration files and can patch optional mod configuration files. Treat changes as operationally sensitive.

## Do not report as security issues

- Normal gameplay balance changes.
- Expected next-reboot profile rotation.

## Report privately

- Path traversal risks.
- Unsafe file write behavior.
- Ability to overwrite files outside intended server/mod paths.
- Malicious command execution.

Until a public security channel exists, report privately to the repository owner.
