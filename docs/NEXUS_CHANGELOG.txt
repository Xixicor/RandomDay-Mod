v0.3.15-alpha

- Renamed the public mod package to RandomDay.
- Updated the install folder to RandomDay/.
- Removed bundled third-party mod folders from the clean release package.
- Removed duplicate lowercase loader file.

- Kept safety rules enforced in generated profiles.
- Kept optional adapter support for DynamicSpawns, Lethal Lasers, and ANTE Up.
