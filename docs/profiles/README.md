# RandomDay Profile Reference

Version: `v0.3.15-alpha`

This reference lists every playstyle family and every day in the family-based 17-day rotation.

The runtime applies one family at a time. The selected family stays active through Day 17, and each reboot advances one day inside that family.

## Families
