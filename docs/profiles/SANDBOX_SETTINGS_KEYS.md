# SandboxSettings.ini Key Reference

These are the sandbox keys emitted by RandomDay profiles in canonical output order.

| Order | Key | Category | Meaning |
|---:|---|---|---|
| 1 | `GameDifficulty` | Difficulty | Overall game difficulty level. |
| 2 | `HardcoreMode` | Safety | Forced off by RandomDay. |
| 3 | `LootRespawnEnabled` | Loot | Controls whether loot respawn is enabled. |
| 4 | `PowerSocketsOffAtNight` | Environment | Controls night power socket behavior. |
| 5 | `DayNightCycleState` | Time | Controls day/night mode. |
| 6 | `DayNightCycleSpeedMultiplier` | Time | Controls cycle speed. |
| 7 | `WeatherFrequency` | Environment | Controls weather frequency. |
| 8 | `SinkRefillRate` | Survival | Controls sink refill rate. |
| 9 | `FoodSpoilSpeedMultiplier` | Survival | Controls food spoilage speed. |
| 10 | `RefrigerationEffectivenessMultiplier` | Survival | Controls refrigerator effectiveness. |
| 11 | `StorageByTag` | Utility | Controls storage tag behavior. |
| 12 | `StructuralSupportLimit` | Safety | Forced to 0 by RandomDay. |
| 13 | `BridgeSupports` | Safety | Forced to 0 by RandomDay. |
| 14 | `HomeWorlds` | World | Sandbox world setting. |
| 15 | `TaintedSinkWater` | Survival | Controls sink water contamination. |
| 16 | `RadiationDealsDamage` | Hazards | Controls whether radiation deals damage. |
| 17 | `InvisibleRadiation` | Hazards | Controls invisible radiation behavior. |
| 18 | `EnemySpawnRate` | Combat | Enemy spawn pressure. |
| 19 | `EnemyHealthMultiplier` | Combat | Enemy health multiplier. |
| 20 | `EnemyPlayerDamageMultiplier` | Combat | Enemy damage to players. |
| 21 | `EnemyDeployableDamageMultiplier` | Combat | Enemy damage to deployables. |
| 22 | `DetectionSpeedMultiplier` | Combat | Enemy detection speed. |
| 23 | `EnemyAccuracy` | Combat | Enemy accuracy level. |
| 24 | `DamageToAlliesMultiplier` | Combat | Damage to allies multiplier. |
| 25 | `HungerSpeedMultiplier` | Survival | Hunger speed multiplier. |
| 26 | `ThirstSpeedMultiplier` | Survival | Thirst speed multiplier. |
| 27 | `FatigueSpeedMultiplier` | Survival | Fatigue speed multiplier. |
| 28 | `ContinenceSpeedMultiplier` | Survival | Continence speed multiplier. |
| 29 | `BonusPerkPoints` | Progression | Bonus perk points. |
| 30 | `PlayerXPGainMultiplier` | Progression | Player XP gain multiplier. |
| 31 | `ItemStackSizeMultiplier` | Loot | Item stack size multiplier. |
| 32 | `ItemWeightMultiplier` | Loot | Item weight multiplier. |
| 33 | `ItemDurabilityMultiplier` | Loot | Item durability multiplier. |
| 34 | `DurabilityLossOnDeathMultiplier` | Death | Durability loss on death. |

| 36 | `AllowRecipeSharing` | Utility | Recipe sharing enabled. |
| 37 | `AllowPagers` | Utility | Pager setting. |
| 38 | `AllowTransmog` | Utility | Transmog enabled. |
| 39 | `DisableResearchMinigame` | Utility | Research minigame toggle. |
| 40 | `DeathPenalties` | Death | Death penalty level. |
| 41 | `FirstTimeStartingWeapon` | Utility | First-time starting weapon. |
| 42 | `HostAccessPlayerCorpses` | Utility | Host corpse access. |
| 43 | `AllowCharacterReset` | Utility | Character reset enabled. |
| 44 | `BaseInventorySize` | Utility | Base inventory slots. |
| 45 | `PlayerFurnitureDestruction` | Utility | Player furniture destruction. |
| 46 | `AllowIronMode` | Safety | Forced off by RandomDay. |
